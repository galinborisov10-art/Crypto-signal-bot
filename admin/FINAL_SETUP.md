### 4. 💾 **Backup System** - Автоматичен

Създаден скрипт: `admin/backup.sh`

```bash
#!/bin/bash
# Ежедневни backups на:
- bot_stats.json
- credentials.json
- admin_password.json
# Съхранение: 30 дни
# Локация: admin/backups/
```

**Как да го стартираш:**
```bash
./admin/backup.sh
```

**Как да го автоматизираш (cron):**
```bash
# Всеки ден в 02:00
0 2 * * * /workspaces/Crypto-signal-bot/admin/backup.sh
```
